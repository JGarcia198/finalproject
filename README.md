# Anecdotal Notes App

A full-stack web application that allows teachers to create, view, and manage anecdotal notes for students.

## Tech Stack
- React (frontend)
- Node.js + Express (backend)
- PostgreSQL (database)

## Planned Features
- Student CRUD
- Anecdotal notes per student
- Optional teacher notification list
- Secure environment variables with .env

Project initialized.
